Imágenes

Texto alternativo

El texto alternativo es una propiedad en la etiqueta <img>. Esta propiedad, aparte de que mostrará el texto que incluyamos en el caso de no poder cargar la imagen, a las personas usuarias de lector de pantalla les será imprescindible para poder percibir el significado de la imagen.

<img src="" alt="Texto alternativo">

En el caso de tener la página en varios idiomas, es importante traducir también el texto alternativo de las imágenes.

El texto alternativo no es el nombre de imagen o pie de imagen, su finalidad tampoco es la de ser contenedor de palabras clave para el posicionamiento SEO.

No hay un límite de caracteres estandarizado, pero se recomienda no superar los 140 caracteres.

Imágenes decorativas

Las imágenes decorativas son aquellas que si prescindimos de ellas en nuestro entorno, nuestro contenido no pierde sentido.

Para que las tecnologías de asistencia puedan ignorar este tipo de imágenes, es necesario que especifiquemos la propiedad alt con las comillas vacías.

<img src="" alt="">

De lo contrario, reconocerá que hay una imagen y solo anunciará su existencia, el usuario no sabrá que contiene. Esto puede provocar frustración al usuario y sensación de entorno no accesible.

Imágenes informativas

Las imágenes informativas son aquellas que aportan información a nuestro contenido y de las cuales no podemos prescindir.

Para este tipo de imágenes es necesario incluir el texto alternativo. Este debe describir la imagen, lo que se percibe visualmente.

Imágenes de texto

En el caso de las imágenes de texto, si este texto contiene información relevante y esta no se encuentra ya en el contenido, se deberá incluir una transcripción literal del texto. Si, por el contrario, esta información ya la podemos encontrar dentro del contenido y esta ya es perceptible y sin pérdidas de información, podemos considerar esta imagen como decorativa.

Antes que utilizar una imagen de texto, lo ideal sería que esta fuera a través de código, una imagen de fondo (decorativa) y el código sobrepuesto. De esta forma, las personas con baja visión podrían personalizar el texto a su conveniencia para poder percibirlo, también sería perceptible para los lectores de pantalla.

Imágenes como enlace

Las imágenes que tienen alguna funcionalidad como enlaces, áreas activas de mapas de imagen o botones de tipo imagen en los formularios, deben tener un texto alternativo que identifica su propósito.

Imágenes agrupadas

En el caso de tener un grupo de imágenes que transmiten una única información, se debe proporcionar el texto alternativo en una sola de las imágenes y describir a todo el grupo, quedando el resto de imágenes como decorativas.
