Uso del color

Asegurar que todos los usuarios pueden acceder a la información que se transmite por el color.

No se debe emplear el color (del texto, de fondo o de otros elementos) como único medio para transmitir información.

Los enlaces no se deben diferenciar del texto que los rodea únicamente mediante el color o al menos tienen un contraste de 3:1 con el texto que los rodea y se deben emplear pistas visuales adicionales cuando reciben el foco.

En las imágenes no se debe emplear el color como único medio visual para transmitir información.
